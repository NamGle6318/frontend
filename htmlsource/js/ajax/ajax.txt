동기식과 비동기식의 차이
동기식 : 요청 -> 응답 -> 실행
비동기식 : 요청 -> 응답이 오던말던 실행
    - 지도 : 지도를 키자마자 대한민국 전체를 주는게 아닌 해당 지점으로 이동할때마다 정보를 받음


Ajax(Asynchroronous JavaScript and XML) : 
- 비동기식 자바스크립트와 XML(JSON) = {자바스크립트 객체}
